# websocket-secured-chat
SpringBoot aplication with secured websocket chat

## Use following URLs by accessing websocket chat client:
* [Authenticated Chat](http://localhost:9090/api/auth/auth-index.html)
* [Non Authenticated Chat](http://localhost:9090/api/index.html)
* [Swagger API docs](http://localhost:9090/api/swagger-ui.html)
* [H2 Console](http://localhost:9090/api/h2-console)
